# **API Testing**

### 1. What is the difference between 201 and 204 Status Code?
**Answer:**
- **201**: Created (response body returned).
- **204**: No Content (successful but no response body).

### 2. What is the difference between 401 and 403 Status Code?
**Answer:**
- **401**: Unauthorized (invalid credentials).
- **403**: Forbidden (user lacks permission).

### 3. What are the components of an API Request?
**Answer:** Request method, headers, body, query parameters, and endpoint URL.

### 4. What is the difference between Query Parameters and Path Parameters?
**Answer:**
- **Query Parameters**: `?id=10` (used for filtering).
- **Path Parameters**: `/users/10` (used for resource identification).
